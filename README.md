KnobV0 is a simple project that has a microcontroller and a rotary encoder. It can be connected to a pc using Bluethooth or USB. For now it can control the volume and mute/unmute. In future it will be fully configurable thanks to the WiFi support that allows us to make it have a custom WebUI to configure everything in one place.
